# ecommerce-backend

This folder contains the backend split into:

- `api/` for the Laravel API workspace
- `legacy-php/` for the original PHP storefront/admin code preserved during migration

## Suggested next steps

1. Continue expanding the Laravel API in `api/`
2. Move remaining cart, checkout, auth, and admin features out of `legacy-php/`
3. Point the Next.js frontend repo at the Laravel API base URL
